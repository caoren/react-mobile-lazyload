# react-mobile-lazyload

> A React components named Lazyload, used in the mobile page.

## Usage
```js
<LazyLoad partial placeholder={(<img src="" />)}>
    <img src=".." />
</LazyLoad>
```

## Options

#### placeholder
Type: `React.node`

Default value: `<div>`

Note: placehoder element.

## partial
Type: `React.bool`

Default value: `false`
